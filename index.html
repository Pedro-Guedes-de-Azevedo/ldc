De que se trata o projeto?

  O projeto aqui tratado, Leitor de cédulas, tem por objetivo auxiliar pessoas com problemas ou deficiências visuais, com uso de inteligência artificial, a identificar cédulas de dinheiro brasileiro.

  Nas notas brasileiras, faltam elementos não visuais identificadores de cada dinheiro, como poderia ser o caso de relevos diferentes para cada valor. Esta falta se trata, para aqueles com baixa/nenhuma capacidade de enxergar, de um obstáculo. O app aqui referido busca fazer o usuário desviar desse obstáculo por meio de uma tecnologia assistiva, que permite com que ele faça uso da audição no lugar da visão para ser capaz de reconhecer valores das notas, uma vez que o software fará uso dos autofalantes do dispositivo para informar as notas.



Qual a inspiração?

  Um dos criadores do projeto tem uma pessoa muito chegada que tem deficiência visual. Devido ao contato muito próximo entre essas duas pessoas no dia-a-dia, foi observado o problema de descobrir o valor de uma nota sem conseguir vê-la.

  A partir disso, o criador fez a proposta ao grupo de colegas, que buscava boas idéias para auxiliar a vida das pessoas pelo uso da tecnologia informática: um app que conseguisse identificar notas para aqueles que não o pudessem.
